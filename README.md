## 🚀 Tính năng nổi bật

- ✅ **Quản lý công, lương, ca làm, thưởng phạt, xuất báo cáo...**

- ✅ **Không cần đăng ký tài khoản (bản Free)**
- ✅ **Lưu trữ dữ liệu trên thiết bị hoặc đồng bộ lên server (bản Pro/Business)**
